# Bilder ersetzen

Diese Dateien sind **Platzhalter** (SVG) und sollten durch echte Fotos ersetzt werden.

| Datei | Verwendung | Empfohlene Groesse |
|-------|------------|--------------------|
| `hero.svg` | Hero-Bild (Startseite, rechts) | Hochformat, ca. 1200 x 1500 px |
| `studio.svg` | "Ueber uns" / Studio-Atmosphaere | ca. 1200 x 1040 px |
| `og.svg` | Social-Preview (OpenGraph) | 1200 x 630 px |

**Hinweis:** Beim Austausch gegen JPG/WebP bitte die Dateinamen-Endung in den
Komponenten anpassen (`/components/sections/Hero.tsx`, `About.tsx`) bzw. in
`app/layout.tsx` (OpenGraph-Bild).
